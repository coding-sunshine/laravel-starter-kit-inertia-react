🚂 Coal Dispatch Flow Journey

Mine → Road Transport → Railway Siding → Rake Loading → Weighment → RR Generation

📍 PHASE 1: MINE OPERATIONS
1
Vehicle Authorization & RFID Entry
All vehicles are pre-authorized and GPS-enabled. Entry allowed only after system validation.
📊 DATA TO CAPTURE:
🚛 Vehicle Number
🔖 RFID Tag
⏰ Entry Timestamp
📡 GPS Coordinates
2
Gate Pass / Loading Slip Issue
Vehicle must handover previous trip gate pass. New gate pass issued only after verification.
🎫 Previous trip gate pass verification
⚖️ Tare weight taken at entry (or fortnightly)
📊 DATA TO CAPTURE:
🎫 Previous Gate Pass Number
📝 New Gate Pass Number
⚖️ Tare Weight (kg)
⏰ Issue Timestamp
3
Movement to Loading Point
Vehicle proceeds to coal loading point and waits in queue.
🚛
4
Coal Loading at Mine
Coal loaded by operators. No overloading/underloading - strictly as per carrying capacity.
📊 DATA TO CAPTURE:
👷 Loader Operator ID
🏗️ Loader Equipment ID
⏰ Loading Start Time
⏰ Loading End Time
5
Weighbridge & Weighment
Vehicle moves to weighbridge. If overloaded, returns to stockyard to drop excess load.
📊 DATA TO CAPTURE:
⚖️ Gross Weight (kg)
⚖️ Tare Weight (kg)
⚖️ Net Weight (kg)
📄 Weighment Slip Number
⏰ Weighment Timestamp
⚠️ VALIDATION: If overloaded → Go back to stockyard → Re-weigh
6
JIMMS Challan Generation
Mining challan generated based on weighment slip and approved route.
📊 DATA TO CAPTURE:
📋 JIMMS Challan Number
🛣️ Approved Route
⏰ Challan Issue Time
7
Dust Suppression & Safety
Vehicle passes through water sprinklers. Entire body covered with tarpaulin.
💧 Water sprinkling
🛡️ Tarpaulin covering
8
Exit Inspection & Verification
Physical inspection at exit point. Verify weighment slip, challan, tarpaulin, and no spillage risk.
✅ Weighment slip verified
✅ Mining challan verified
✅ Tarpaulin covering checked
✅ No spillage risk confirmed
📊 DATA TO CAPTURE:
👮 Inspector ID
⏰ Exit Timestamp
✅ Verification Status
🛣️ PHASE 2: ROAD TRANSIT MONITORING
9
GPS-Based Transit Monitoring
Live vehicle tracking from control room. Monitor route, stoppages, speed, and delays.
🚛💨
📊 DATA TO CAPTURE:
📍 Real-time GPS Coordinates
🛣️ Route Adherence Status
🚦 Stoppage Events
📏 Deviation Distance (if any)
⚡ Speed Data
10
Road Patrolling
Dedicated patrol teams monitor vehicle movement. Ensure timely arrival, no pilferage, no deviations.
⚠️ ALERT CONDITIONS: Route deviation, Unscheduled stops, Speed anomalies
🏭 PHASE 3: RAILWAY SIDING COAL RECEIPT
11
Coal Receipt at Railway Siding
Loaded trucks go directly to weighbridge at siding. Verify same load received.
📊 DATA TO CAPTURE:
⚖️ Siding Gross Weight
⏰ Receipt Timestamp
🏭 Siding Name (Pakur/Dumka/Kurwa)
👷 WB Operator ID
📦 Stock Update (+)
⚠️ VALIDATION: Cannot receive without dispatch reference from mine
12
Unloaded Truck Weighment
After unloading, truck moves to weighbridge for tare weight. Ensures complete unloading.
📊 DATA TO CAPTURE:
⚖️ Tare Weight at Siding
⏰ Tare Timestamp
✅ Trip Completion Status
📋 PHASE 4: RAKE INDENT & PLANNING
13
Indent for Rake Placement
Siding In-Charge checks available stock and raises indent to Railways.
📦 Check siding stock availability
🎯 Determine target rake quantity
📊 DATA TO CAPTURE:
📋 Indent Number
⏰ Indent Date & Time
🏭 Siding Name
🎯 Target Quantity (MT)
📦 Available Stock (MT)
🔢 Railway Reference Number
⚠️ IMPORTANT: Indent is STOCK-DRIVEN, not time-driven. Cannot raise indent without sufficient stock!
14
Rake Availability & Placement
Railways place rake based on availability after receiving indent.
📊 DATA TO CAPTURE:
🚂 Rake Number
⏰ Rake Placement Timestamp
🔢 Total Wagons
🔧 PHASE 5: TXR INSPECTION & LOADING
15
TXR Inspection (Before Loading)
Train Examiner inspection by Railway staff. Check all wagon doors and conditions.
📊 DATA TO CAPTURE:
⏰ TXR Start Time
⏰ TXR End Time
⏱️ TXR Duration (Auto-calculated)
✅ Fit Wagon Count
❌ Unfit Wagon Count
16
Marking of Unfit Wagons
Unfit wagons marked clearly with radium reflectors/red lights (night) or red flags (day).
📊 DATA TO CAPTURE (Per Unfit Wagon):
🚃 Wagon Number
❌ Reason for Unfit
🏷️ Marking Method (Flag/Reflector/Light)
📋 Railway Memo Reference
⚠️ AUTO-EXCLUSION: These wagons must be excluded from loading screens!
17
⏰ CRITICAL: Loading Time Starts!
Loading time starts from rake placement. Maximum FREE time: 3 HOURS.
⏱️ FREE TIME REMAINING: 3:00:00
🚨 PENALTY RULE: Even 5 minutes extra beyond 3 hours = Railway charges 1 FULL HOUR penalty!
18
Wagon Loading Process
Loader operators load wagons one by one using inbuilt load cells. No overloading/underloading.
📊 DATA TO CAPTURE (Per Wagon):
🚃 Wagon Number
🏗️ Loader ID
👷 Loader Operator ID
📦 CC Capacity (MT)
⚖️ Loaded Quantity (MT)
⏰ Loading Timestamp
⚠️ AUTO-ALERT: System must flag overload/underload risk in real-time!
19
Post-Loading Inspection by Guard
Railway Guard inspects for coal spillage on tracks and between wagons. Must be cleared before movement.
📊 DATA TO CAPTURE:
⏰ Loading Completion Time
✅ Guard Inspection Status
🧹 Spillage Clearance (Yes/No)
20
Clearance for Movement
After clearance, Guard informs Loco Pilot and gives permission to move rake.
📊 DATA TO CAPTURE:
⏰ Movement Permission Time
👮 Guard ID
✅ Clearance Status
⚖️ PHASE 6: IN-MOTION WEIGHMENT (VERY SENSITIVE!)
21
🚨 In-Motion Weighment
Loco pilot moves rake from loading track to weighment track. CRITICAL PROCESS!
⏱️ Time Limit: 20 minutes (included in 3-hour free time)
🚄 Speed: Must be 5-7 kmph
⚠️ If weighment fails → Take forward → Bring back → Re-weigh
📊 DATA TO CAPTURE (Per Wagon):
🚃 Wagon Number
⚖️ In-Motion Weight (MT)
📄 Weighment Slip Number
⏰ Weighment Timestamp
🚨 EXTREME SENSITIVITY: Wrong speed or jerks can cause wrong weighment & false overloading indication!
22
🔍 Loader vs Weighment Comparison
AUTOMATIC comparison by system. This is your PENALTY PREDICTION engine!
📊 Compare: Loader Quantity vs In-Motion Weight
➖ Calculate: Difference per wagon
🏷️ Classify: Overload / Underload / OK
🚨 Generate: Pre-RR Early Warning Alerts
🎯 KEY OUTPUT: EARLY WARNING DASHBOARD - Predict penalties BEFORE RR arrives!
📄 PHASE 7: RAILWAY RECEIPT (FINAL & BINDING)
23
Generation of Commercial RR
After successful weighment, Commercial Railway Receipt (RR) is generated. THIS IS THE FINAL AUTHORITY DOCUMENT!
⚖️ Charged Weight
⚠️ Overloading / Underloading
⏱️ Time Taken
💰 Demurrage
🚫 Penalties
💵 Freight & GST
24
Receipt of RR & Documents
At Railway Siding, we receive in-motion weighment slip, TXR unfit memo, and forwarding copy.
📊 DATA TO CAPTURE:
📄 RR Number
📤 Upload RR PDF
🤖 Auto-extract RR data (OCR)
✍️ Manual entry fields (backup)
💰 Freight Amount
🚫 Penalty Amount
📊 GST Amount
🔍 RECONCILIATION: Compare RR data with system's loader & weighment data!
25
🔄 End-to-End Reconciliation
Final reconciliation across all stages to ensure data integrity and identify discrepancies.
1️⃣ Mine vs Siding (Road dispatch quantities)
2️⃣ Siding vs Rake (Stock utilization)
3️⃣ Rake vs Weighment (Loading accuracy)
4️⃣ Weighment vs RR (Railway official record)
5️⃣ RR vs Power Plant receipt (Final delivery)
