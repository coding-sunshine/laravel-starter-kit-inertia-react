# Keyboard shortcuts

You can use these shortcuts to navigate and use the app more quickly.

| Shortcut | Action |
|----------|--------|
| **⌘K** (Mac) or **Ctrl+K** (Windows/Linux) | Open the command palette (search, recent pages, navigation, fleet assistant) |
| **Tab** | Move focus to the next focusable element |
| **Shift+Tab** | Move focus to the previous focusable element |
| **Enter** or **Space** | Activate the focused link or button |
| **Escape** | Close the command palette, a modal, or a dialog |

## Skip to main content

On every page, the first focusable element is a “Skip to main content” link. When you press **Tab** after the page loads, focus goes to this link; pressing **Enter** moves focus to the main content and skips the main navigation. This helps keyboard and screen-reader users get to the page content quickly.

## Command palette

Press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux) to open the command palette. From there you can:

- Type to search (users, posts, help articles, changelog)
- Choose a navigation item (Dashboard, Billing, Help, etc.)
- Open Settings or log out

The palette shows the same shortcuts at the bottom (Escape to close, ⌘K / Ctrl+K to open again).
