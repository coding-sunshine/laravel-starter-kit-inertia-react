# User Guide

Welcome to the User Guide. This documentation helps you understand and use the features of this application.

## Table of Contents

- [Authentication](./authentication/README.md) - Account creation, login, and security features
- [Settings](./settings/README.md) - Profile, password, 2FA, appearance, Level & achievements, data export

## Getting Started

If you're new to the application, start with the [Authentication](./authentication/README.md) section to learn how to create an account and sign in. Use [Settings](./settings/README.md) to manage your profile, security, and preferences. If you manage users and access, see [Managing roles and permissions](./roles-and-permissions.md).
