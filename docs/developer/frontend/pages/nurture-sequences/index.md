# nurture-sequences/index

## Purpose

Lists all nurture sequences with enrollment counts, step counts, and active status. Links to create new sequences.

## Location

`resources/js/pages/nurture-sequences/index.tsx`

## Route

`GET /nurture-sequences` (`nurture-sequences.index`)

## Props

- `sequences` — Paginated list of NurtureSequence models with enrollments_count and steps_count

## Related Components

- **Controller**: `NurtureSequenceController`
