# Pages

Inertia pages are React components that receive data from Laravel controllers. They live in `resources/js/pages/`.

## Available Pages

| Page | Route | Documented |
|------|-------|------------|
| welcome | `/` | ❌ |
| dashboard | `/dashboard` | ❌ |
| session/create | `/login` | ❌ |
| user/create | `/register` | ❌ |
| user-profile/edit | `/settings/profile` | ❌ |
| user-password/edit | `/settings/password` | ❌ |
| appearance/update | `/settings/appearance` | ❌ |
| user-two-factor-authentication/show | `/settings/two-factor` | ❌ |

> **Note**: Individual page documentation will be added here as they are documented.
