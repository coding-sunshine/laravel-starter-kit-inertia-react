# lead-generation/index

## Purpose

Lead generation dashboard showing stats (total leads, hot leads, engagement, active sequences), recent leads, active nurture sequences, and AI tool quick-links.

## Location

`resources/js/pages/lead-generation/index.tsx`

## Route

`GET /lead-generation` (`lead-generation.index`)

## Props

- `stats` — LeadStats (total_leads, hot_leads, engagement_today, active_sequences)
- `recent_leads` — Array of recent lead contacts
- `active_sequences` — Array of active nurture sequences

## Related Components

- **Controller**: `LeadGenerationController`
