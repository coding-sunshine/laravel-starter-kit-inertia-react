# Empty state illustrations

SVG assets for empty states. Use with `EmptyState` or `FleetEmptyState` via the `illustration` prop.

| File | Use case |
|------|-----------|
| `vehicles.svg` | No vehicles, add first vehicle |
| `fleet.svg` | General fleet empty (e.g. no routes, no trips) |
| `data-empty.svg` | No data, reports, or list data |

Example: `illustration="/images/empty/vehicles.svg"`

Icon set: Lucide is retained project-wide.
